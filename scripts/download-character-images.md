# Download Character Images Script

## Quick Setup Instructions

Since we need actual anime character images, here are the best sources:

### Option 1: Use Wikimedia Commons (Free & Legal)
Many anime characters have images on Wikimedia Commons under fair use.

### Option 2: Use Official Anime Databases
- **MyAnimeList**: https://myanimelist.net/
- **AniList**: https://anilist.co/
- **Anime-Planet**: https://www.anime-planet.com/

### Option 3: Manual Download (Recommended for Quality)

For each character, search Google Images with these exact queries:

1. **Goku**: "goku dragon ball z official art"
2. **Naruto**: "naruto uzumaki official art"
3. **Luffy**: "monkey d luffy official art"
4. **Light Yagami**: "light yagami death note official art"
5. **Saitama**: "saitama one punch man official art"
6. **Levi**: "levi ackerman attack on titan official art"
7. **Sailor Moon**: "sailor moon usagi official art"
8. **Mikasa**: "mikasa ackerman attack on titan official art"
9. **Nezuko**: "nezuko kamado demon slayer official art"
10. **Anya**: "anya forger spy x family official art"
11. **Power**: "power chainsaw man official art"
12. **Titan**: "smiling titan attack on titan"
13. **Chopper**: "tony tony chopper one piece official art"
14. **Mr. Satan**: "mr satan hercule dragon ball z"
15. **Pikachu**: "pikachu pokemon official art"
16. **Death Note**: "death note book"
17. **Dragon Balls**: "dragon balls dragon ball z"
18. **One Piece**: "one piece treasure"

### Quick Download Steps:

1. Search for each character
2. Find a high-quality square image (or crop to square)
3. Save as JPG with the exact filename from the list above
4. Place in `public/characters/` folder
5. Refresh the quiz page - images will load automatically!

### Temporary Solution

Until you add real images, the quiz will show placeholder emojis. The functionality works perfectly - just needs the visual assets!
